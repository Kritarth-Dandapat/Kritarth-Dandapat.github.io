Marine Guardian: Ship Detection in Satellite Imagery

Computer Vision & Deep Learning Project | CSE 473 Final Project

Project Overview

The Marine Guardian project aims to develop a model for detecting ships in satellite imagery, crucial for maritime security, environmental monitoring, and naval operations. The goal was to establish baseline performance using existing models and then explore first principles of computer vision for faster, real-time detection.

Technical Challenge

The project addressed the trade-off between accuracy and speed in ship detection. While well-defined CNNs provide good accuracy, they lack the speed needed for live monitoring. The approach was inspired by face detection, where Haar filters provide faster detection than CNNs.

Dataset & Methodology

  • Dataset: Ships in Satellite Imagery from Kaggle (1000 ship images, 3000 no-ship images)
  • Challenge: Dataset labeling inconsistencies with partial ships classified as "no-ship"
  • Approach: Combined transfer learning with first principles of computer vision

Technical Implementation

  • Baseline Models: Basic CNN, EfficientNetB0, MobileNetV2
  • Transfer Learning: Used pre-trained EfficientNet as encoder with custom decoder
  • Unsupervised Learning: ResNet50 + KMeans clustering for dataset validation
  • First Principles: Roundness-based classification using geometric properties

Key Results

  • MobileNetV2: 98.72% accuracy, 99.31% validation accuracy (best performance)
  • EfficientNetB0: 74.25% accuracy (affected by dataset issues)
  • Roundness Algorithm: Detected 2234 ship images (close to expected 2000)
  • Speed Optimization: Fast detection using geometric properties for real-time monitoring

Technical Innovations

  • Roundness Calculation: Developed algorithm using area-to-perimeter ratio for shape classification
  • Dataset Validation: Uncovered and addressed labeling inconsistencies
  • Hybrid Approach: Combined deep learning with traditional computer vision techniques
  • Real-time Processing: Optimized for live satellite imagery monitoring

Technologies Used

  • Python with OpenCV for image processing
  • TensorFlow/Keras for deep learning models
  • EfficientNet, MobileNetV2, ResNet50 architectures
  • KMeans clustering for unsupervised learning
  • Geometric algorithms for roundness calculation
  • Google Colab for GPU-accelerated training

Impact & Applications

This project demonstrates the importance of dataset validation and the effectiveness of combining modern deep learning with traditional computer vision techniques. The roundness-based approach provides a fast, efficient method for ship detection that can be deployed in real-time maritime monitoring systems.

Lessons Learned

  • Thorough dataset validation is crucial before model development
  • Simple geometric solutions can outperform complex neural networks in specific applications
  • Documentation and organization of experimental results is essential
  • First principles thinking can lead to unexpected but effective solutions
